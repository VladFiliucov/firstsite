== README


Social network application by treahouse tutorial.

Rails 4.1

Twitter Bootstrap for design.
Devise for authentication.
Simple_form for better forms.
Gravatar for avatars.

In test uses sqlite3 database
In production uses postgress database

application is deployed to Heroku:
http://filiucov.herokuapp.com/
